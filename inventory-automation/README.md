# Inventory Automation (PowerShell)

## Overview
Developed a PowerShell-based automation tool to collect system, network, and hardware inventory across multiple machines, reducing manual effort and improving accuracy

## Features
- Collects system name and login ID
- Gets IP and MAC address
- Fetches hardware details (RAM, CPU, Storage)
- Detects installed software (Office, Antivirus)
- Exports data into CSV format

## Usage
Run the script on each system:
.\inventory.ps1

## Output
inventory.csv file with system details
